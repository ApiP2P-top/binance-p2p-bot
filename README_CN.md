## 🚀 核心功能

### 1. 自动出价与实时调价 (智能狙击排名)
* **竞品监控：** 实时追踪 P2P 广告列表中竞争对手的价格变化。
* **自动排名：** 根据预设策略（例如：永远比第一名价格优 $0.01）自动调整您的广告价格，确保您的广告始终占据优质曝光位。
* **多目标竞价逻辑：** 输入特定商家的 P2P 用户名，算法将锁定最佳排名位并在其基础上应用您的预设加价。
* **安全区间保护：** 允许设置价格上限和下限，防止在极端行情下发生亏本交易。严守您的最高/最低限价红线。

### 2. 高波动资产快速同步 & 稳定币批量管理
* **现货市场同步：** 针对 BTC、ETH、SOL 等价格变化极快的资产，机器人会根据最新交易所现货价格乘以您的预设比例，瞬间更新您的 P2P 价格。
* **稳定币批量操作：** 对于 USDT/USDC/FDUSD 等资产，支持一键批量上架/下架数十个广告。

### 3. 多平台支持
* 无缝兼容 **币安** 和 **Bybit** 商家后台，实现高效的跨平台广告管理。主动处理平台特定的业务逻辑（例如处理 Bybit 较低的可用资金门槛要求）。

### 4. 自动化订单流与实时库存更新
* 全自动处理广告的上架/下架及基础的订单流程。
* **库存感知：** 以毫秒级速度监控您的现货与资金账户余额。交易完成后瞬间感知余额变化，平滑更新所有在线 P2P 广告的可用数量。

### 5. 专为商家优化的操作界面
* 通过 API 聚合专为高频交易打造。界面深度优化，适合管理大量买卖订单的专业用户。

## ⏱️ 核心技术参数与底层机制

为了在追求高频交易效率与严格的账号风控之间取得平衡，我们的线程模型内置了严谨的执行周期与休眠策略：

* **并发广告轮询：** 主引擎每 **3000 毫秒（3秒）** 完整查询并对比激活状态的广告队列。
* **竞价网络扫描器：** 为了追踪对手，底层网络每 **2000 毫秒（2秒）** 进行深度抓取，快速拦截市场变动。
* **输入防抖阈值：** 内置 **1500 毫秒（1.5秒）** 的防抖机制，防止产生无意义的 API 请求垃圾。
* **安全速率限制 (风控熔断器)：** 部署本地多任务锁队列，如果触发全局 API 流量限制拦截，将自动触发最高 300 秒的风控熔断保护机制。
